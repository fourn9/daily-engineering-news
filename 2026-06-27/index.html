Announcing TypeScript 7.0 RC

🔥話題🆕新リリース
devblogs.microsoft.com｜2026-06-18

TypeScript 7.0 のリリース候補が公開。コンパイラを Go に移植した Project Corsa が本流の tsc に統合され、型チェック / LSP 応答が概ね 約10倍 高速化。VS Code（約150万行）のフルビルドは 77.8秒 → 7.5秒。挙動は port（書き直しではない）のため、型推論セマンティクスは従来と完全互換。安定版 GA は約1ヶ月後を予定。

Run isolated sandboxes with full lifecycle control: AWS Lambda introduces MicroVMs

🆕新リリース📐設計論
aws.amazon.com｜2026-06-22

⚠セキュリティ🔥話題
thehackernews.com／Novee Security｜2026-06

Novee Security が公開した CI/CD 系の体系的脆弱性 "Cordyceps"。GitHub Actions のワークフロー設定における過剰権限、command injection、artifact poisoning、broken auth ロジック等を組み合わせ、未認証の外部攻撃者がPR経由で他者リポジトリの CI を乗っ取れる 攻撃チェーン群。約3万リポジトリを走査し、300以上で完全に実行可能な攻撃を確認。Microsoft / Google / Apache / Cloudflare / Python Software Foundation 等で修正済。Novee の元記事 も併読推奨。

LLM が "ユーザーの曖昧な説明" にすぐ同調して結論を出してしまう user-driven sycophancy 問題に対し、(1) 初期記述の曖昧さを推定 → (2) 競合仮説を生成 → (3) 焦点を絞った確認質問 → (4) 回答ごとに仮説確率を更新 → (5) ある仮説が明確に優位になるまで継続、という Solution Investigator Agent を提案。機械 / 電気 / 油圧の技術フォーラム実問題でベンチを構築。
